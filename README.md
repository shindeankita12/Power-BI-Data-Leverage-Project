Data Leverager – Power BI ETL Project :-

Data Sources Used :-

1. Sales Data (Excel files – Jan, Feb, Mar, Apr)
2. Employee Data (Excel file)
3. Web Data (HTML table from web)

Transformations Applied :-

* Removed blank rows and duplicates
* Promoted headers
* Renamed columns
* Changed data types
* Text cleaning using UPPER, LOWER, TRIM, CLEAN
* Created Profit column
* Extracted Year, Month, Quarter from Order Date
* Created Sales Category using Conditional Column
* Added Index columns
* Pivot and Unpivot operations
* Merged Sales and Employee data
* Appended monthly sales files
* Grouped data by Region
* Data profiling and quality checks
* Created Parameters for dynamic folder path
* Refresh simulation for new monthly files

 Tools Used :-

* Power BI Desktop
* Power Query Editor
* Excel
* GitHub

Challenges Faced :-

* Combining multiple Excel files
* Data type errors
* Null values and duplicates
* Merge query errors
* Parameter setup for folder refresh
